# HyperDocs

## Description

Full-stack collaborative document editor built with React and Hono designed for seamless real-time collaboration.
This app provides a rich text editing experience enhanced with artificial intelligence for document summarization and text assistance.

## Features

- **Rich Text Editor:** Feature-rich editor powered by Tiptap with support for headings, lists, blockquotes, code blocks, links, images, text alignment, fonts, colors, and highlighting.
- **Real-Time Collaboration:** Multiple users can edit documents simultaneously with typing indicators and presence avatars powered by Yjs and Hocuspocus.
- **AI Document Summarization:** Generate concise summaries of documents using OpenAI's GPT models.
- **AI Text Assistance:** Get AI-powered suggestions and improvements for selected text.
- **Document Sharing:** Share documents via public links or invite specific users by email with viewer or editor roles.
- **Role-Based Access Control:** Document owners can manage member permissions and control who can view or edit their documents.
- **Document Management:** Organize documents with starring, search functionality, and trash with restore capability.
- **Comments:** Add and manage comments on documents for feedback and discussion.
- **User Authentication:** Secure login via Google, GitHub and Facebook providers, powered by Better Auth.
- **Zoom and Print:** Adjust document zoom levels and print documents directly from the editor.
